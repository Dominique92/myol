EDITEUR: Modifier une ligne ou un polygone

Cliquer sur le bouton ✥ (qui bleuit)

Cliquer (curseur 👆) sur une ligne ou un polygone bleu (modifiable) pour le sélectionner (elle rougit)


Cliquer sur un sommet et glisser pour le déplacer

Cliquer le long d'un segment puis glisser pour ajouter un sommet au milieu d'un segment

Alt+cliquer sur un sommet pour le supprimer

Shift+cliquer sur une ligne pour en inverser la direction


Ctrl+cliquer sur une ligne pour la couper en deux

Déplacer l'extrémité d'une ligne jusqu'à l'extrémité d'une autre pour les fusionner


Ctrl+cliquer sur un sommet ou un segment d'un polygone pour le transformer en ligne

Puis déplacer l'uine des 2 extrémités (rond rouge)


Joindre les 2 extrémités d'une ligne pour la transformer en polygone

Un polygone entièrement inclus dans un autre crée un trou


Ctrl+Alt+cliquer sur un de ses segment pour supprimer une ligne ou un polygone

EDITEUR: Créer un polygone

Cliquer sur le bouton ⏢ (qui bleuit)

Cliquer sur l'emplacement du début

Puis sur chaque sommet à créer

Cliquer sur un tracé existant (non modifiable, en noir) pouy s'y coller" et le suivre

Double cliquer sur le dernier sommet pour terminer
